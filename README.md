# 📉 Customer Churn Prediction

A machine learning project that predicts customer churn using historical data to help businesses improve retention.

## 🚀 Features

* Data preprocessing & EDA
* Multiple ML models (Logistic Regression, Random Forest )
* Model evaluation (Accuracy, ROC-AUC)

## 🛠️ Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## 📌 Outcome

Achieved ~85–90% accuracy in predicting customer churn
